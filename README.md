# 🚚 העגלה של אלה

משחק פוד-טראק כיפי וצבעוני לילדים — בנוי כ-PWA (אפליקציית ווב) ב-HTML/CSS/JavaScript טהור.
**בלי התקנה, בלי אינטרנט נדרש אחרי טעינה ראשונה, בלי פרסומות, בלי רכישות.** 100% בטוח לילדים.

---

## 🎮 איך משחקים
- לקוחות חמודים עומדים מול העגלה עם בועה שמראה מה הם רוצים (🥤 מילקשייק / 🍔 המבורגר / 🍕 פיצה).
- לכל לקוח יש **מד-סבלנות** (פס צבע + פרצוף). מקישים על הלקוח כדי להכין את ההזמנה במיני-משחק במגע.
- ככל שמכינים מהר יותר → **טיפ גדול יותר** ויותר כוכבים ⭐.
- לקוח שמתעצבן פשוט אומר "אוף, לא הספקתי!" והולך — **בלי Game Over, בלי עונש**. מיד מגיע לקוח חדש.
- אוספים מטבעות 🪙 וקונים **שדרוגים** בחנות 🛒 (יותר לקוחות, סבלנות, טיפים, רקעים, עיצוב עגלה ועוד).
- יש גם מצב **🎨 יצירה חופשית** — מכינים אוכל בכיף בלי לחץ ובלי לקוחות.
- הכל נשמר אוטומטית במכשיר (גם אחרי סגירה).

## 📂 מבנה הקבצים
```
העגלה של אלה/
├── index.html          ← הקובץ הראשי
├── style.css           ← רקע ועיצוב בסיס
├── manifest.json       ← הגדרות PWA
├── sw.js               ← Service Worker (עבודה אופליין)
├── assets/icon.svg     ← אייקון האפליקציה
├── vendor/phaser.min.js← מנוע המשחק Phaser 3 (מקומי, ללא אינטרנט)
└── js/                 ← קוד המשחק
    ├── audio.js        ← צלילים (Web Audio)
    ├── state.js        ← כלכלה ושמירה (localStorage)
    ├── game.js         ← תשתית Phaser + טעינה
    ├── world.js        ← העולם החי + לקוחות
    └── minigame.js     ← 4 המיני-משחקים + החנות
```
> המשחק בנוי על מנוע **Phaser 3** ששמור מקומית בתיקיית `vendor` — אין שום תלות באינטרנט או ב-CDN.

# 🟢 דרך א' — ההכי קלה (מומלצת)

## 1. בדיקה מהירה במחשב
> שימו לב: כדי ש-Service Worker וה-PWA יעבדו צריך **שרת מקומי** (לא מספיק לפתוח את הקובץ ישירות מהדיסק).

אם מותקן **Python** (קיים כמעט בכל מחשב):
1. פותחים שורת פקודה בתיקיית המשחק.
2. מריצים:
   ```
   python -m http.server 8000
   ```
3. פותחים בדפדפן: `http://localhost:8000`

*(לבדיקה מהירה בלבד אפשר גם פשוט לפתוח את `index.html` בדאבל-קליק — המשחק יעבוד, רק התכונות האופליין של PWA לא ייבדקו כך.)*

## 2. העלאה לאינטרנט וקבלת קישור (חינם)

### אפשרות Netlify (הכי פשוט — גרירה):
1. נכנסים ל-https://app.netlify.com/drop
2. **גוררים את כל התיקייה "העגלה של אלה"** אל החלון.
3. מקבלים תוך שניות קישור כמו `https://ella-cart.netlify.app` — זהו! 🎉

### אפשרות GitHub Pages:
1. פותחים חשבון חינמי ב-https://github.com ויוצרים מאגר (Repository) חדש.
2. מעלים את כל קבצי התיקייה (כפתור **Add file → Upload files**).
3. נכנסים ל-**Settings → Pages**, בוחרים את ענף `main` ותיקייה `/root`, ושומרים.
4. אחרי דקה מקבלים קישור כמו `https://your-name.github.io/ella-cart/`.

## 3. הוספה למסך הבית של האייפד (כמו אפליקציה אמיתית) 📱
1. פותחים את הקישור ב-**Safari** באייפד (חשוב — דווקא Safari).
2. מסובבים את האייפד **לרוחב** (landscape).
3. מקישים על כפתור **שיתוף** (הריבוע עם החץ ⬆️).
4. בוחרים **"הוסף למסך הבית"** (Add to Home Screen).
5. עכשיו יש אייקון על המסך. פותחים אותו → המשחק רץ **במסך מלא, בלי שורת כתובת, וגם בלי אינטרנט** ✨

# 🔵 דרך ב' — App Store אמיתי (אופציונלי בלבד)

רק אם רוצים אפליקציה רשמית בחנות. דורש קצת עבודה ועלות:

**מה צריך:**
- מחשב **Mac** עם **Xcode**.
- חשבון **Apple Developer** (בתשלום שנתי, כ-99$).

**שלבים ברמת כותרות (עוטפים את אותו קוד ווב עם Capacitor):**
1. מתקינים Node.js ואז Capacitor: `npm install @capacitor/core @capacitor/cli`.
2. מאתחלים פרויקט ומגדירים את תיקיית המשחק כ-`webDir`.
3. מוסיפים פלטפורמת iOS: `npx cap add ios`.
4. מעתיקים את הקבצים: `npx cap copy`.
5. פותחים ב-Xcode: `npx cap open ios`, מגדירים אייקון/שם/הרשאות.
6. בודקים על אייפד אמיתי, ואז מעלים דרך **App Store Connect** לאישור אפל.

> לרוב הצרכים — **דרך א' מספיקה לגמרי** ונותנת חוויה כמו אפליקציה, בחינם וללא Mac.
